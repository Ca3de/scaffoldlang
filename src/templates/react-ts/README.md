## 🚀 Quick Start

### Prerequisites

- Node.js 16+ 
- npm or yarn

### Installation

```bash
# Navigate to your project
cd {{projectName}}

# Install dependencies
npm install

# Start development server
npm run dev
```

Your app will be available at `http://localhost:5173`
